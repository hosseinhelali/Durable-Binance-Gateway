## Description
When you need to subscribe to Binance data or other cryptocurrency data source your service must be overcome 
the problems:
* Single point failure
* Scalability

For solving this problems we run multiple node in production mode. Every node dose responsible for part of 
coins when failing a node another node take the node responsibility.
